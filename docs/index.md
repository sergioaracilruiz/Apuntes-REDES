# Instalación física de redes

Apuntes y documentación pública sobre la instalación física de una red local,
basados en estándares profesionales y explicados de forma clara y estructurada.

## Contenidos

- Medios de transmisión
- Cableado estructurado
- Conectores y herramientas
- Ethernet
- Buenas prácticas y seguridad
- Gestión de residuos (RAEE)

## Pensado para:
- Estudiantes de redes
- Docencia
- Consulta técnica
- Autoaprendizaje
